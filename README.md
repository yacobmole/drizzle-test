# Drizzle Relations V2 Example
This project demonstrates a minimal use of Drizzle ORM's Relations V2 features with a simple domain modeled in TypeScript.

## Project Structure
- `src/schema.ts` &mdash; basic table definitions
- `src/db.ts` &mdash; database client configuration and Relations V2 configuration
- `src/index.ts` &mdash; runnable example showcasing relational queries
- `migrations/**` &mdash; SQL migration artifacts generated for the schema

## Prerequisites
- Installed dependencies via `pnpm install`

## Getting Started
1. Install dependencies: `pnpm install`
2. Run Dev: `pnpm dev`
